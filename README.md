# FinanZen
📱 Aplicación móvil Flutter para gestión inteligente de finanzas personales con análisis de ingresos, gastos y optimización del ahorro

## 🚀 Desarrollo del Proyecto

Este proyecto sigue una metodología de desarrollo estructurada con milestones claramente definidos:

### 🎯 Milestones del Proyecto

1. **🏗️ Setup/Inicialización** - Configuración base del proyecto
2. **🚀 MVP Implementation** - Producto mínimo viable  
3. **⚡ Optimización Financiera** - Funcionalidades avanzadas
4. **📊 Gráficas e Info Visual** - Visualizaciones y análisis

### 🛠️ Scripts de Automatización

En el directorio `scripts/` encontrarás herramientas para automatizar la gestión del proyecto:

- **Creación automática de GitHub Projects y Milestones**
- **Asignación automática de issues a milestones**
- **Configuración personalizable para otros proyectos**

Consulta `scripts/README.md` para instrucciones detalladas de uso.

## 🏗️ Arquitectura

El proyecto utiliza **Clean Architecture** con Flutter, organizando el código en capas bien definidas:

- **Presentation Layer** - UI y gestión de estado con BLoC
- **Domain Layer** - Lógica de negocio y entidades
- **Data Layer** - Repositorios y fuentes de datos

## 🔥 Backend

- **Firebase Firestore** - Base de datos NoSQL
- **Firebase Auth** - Autenticación con Google Sign-In
- **Firebase Storage** - Almacenamiento de archivos
- **Firebase Analytics** - Análisis de uso

## 📱 Funcionalidades Principales

### MVP (v1.0)
- ✅ Autenticación y registro de usuarios
- ✅ Gestión de transacciones (ingresos/gastos)
- ✅ Categorización automática y manual
- ✅ Cálculo de margen de ahorro
- ✅ Dashboard financiero básico

### Optimización (v2.0)
- 🔄 Calculadora de eliminación de deudas
- 🔄 Estrategias de optimización inteligente
- 🔄 Análisis con IA personalizado

### Visualización (v3.0)
- 🔄 Gráficas interactivas avanzadas
- 🔄 Filtros y análisis temporal
- 🔄 Reportes exportables

## 🛠️ Stack Tecnológico

- **Frontend**: Flutter + Dart
- **Estado**: flutter_bloc + freezed
- **Backend**: Firebase suite
- **Base de datos**: Cloud Firestore
- **Gráficas**: fl_chart / syncfusion_flutter_charts
- **Testing**: flutter_test + mocktail + bloc_test
